# pitch-detectors
collection of pitch detection algorithms with unified interface

## list of algorithms
1. PraatAC [cpu]
1. PraatCC [cpu]
1. PraatSHS [cpu]
1. Pyin [cpu]
1. Reaper [cpu]
1. Yaapt [cpu]
1. Rapt [cpu]
1. World [cpu]
1. TorchYin [cpu]
1. Crepe [cpu, gpu]
1. TorchCrepe [cpu, gpu]
1. Swipe [cpu, gpu]

## additional features
- robust (vote-based + median) averaging of pitch
- json import/export

## install
```bash
pip install pitch-detectors
```
