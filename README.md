# pitch-detectors
collection of pitch detection algorithms with unified interface

## list of algorithms
1. PraatAC
1. PraatCC
1. PraatSHS
1. Pyin
1. Reaper
1. Yaapt
1. Rapt
1. World
1. TorchYin
1. Crepe **gpu**
1. TorchCrepe **gpu**
1. Swipe **gpu**

## additional features
- robust (vote-based + median) averaging of pitch
- json import/export

## install
```bash
pip install pitch-detectors
```
